# Przydatne linki KSeF (Środowisko Testowe)

## Aplikacja Podatnika (Web) - Generowanie Tokenów
Tutaj zalogujesz się (np. Podpisem Kwalifikowanym lub Profilem Zaufanym) i wygenerujesz `token` autoryzacyjny.
* **URL:** [https://ksef-test.mf.gov.pl/web/](https://ksef-test.mf.gov.pl/web/)
* _Uwaga: Aby wygenerować token, musisz być zalogowany w kontekście firmy i wejść w zakładkę "Uprawnienia"._

## Dokumentacja API (Swagger / OpenAPI v2)
Oficjalna dokumentacja techniczna endpointów API v2.
* **URL:** [https://api-test.ksef.mf.gov.pl/docs/v2/index.html](https://api-test.ksef.mf.gov.pl/docs/v2/index.html)
* **API Base URL (w kodzie):** `https://api-test.ksef.mf.gov.pl/v2`
